# APS指揮台

進階生產排程系統，依產能、物料、交期限制產生最佳化生產排程。

- 原始系統：APS（Advanced Planning and Scheduling）
- 產業分類：智慧製造｜製造管理
- 使用部門：生管、PMC、業務、廠務、資材
- 工作流程：訂單匯入 → 產能/物料檢核 → 排程演算 → 派工單產出 → 執行回饋 → 動態重排
- AI 應用：AI 預測異常插單影響並自動重排、機器學習優化排程規則權重、智慧瓶頸預警
- 主要 KPI：準時交貨率、產能利用率、在製品週轉天數、排程異動次數
- 代表產品：Siemens Opcenter APS、Asprova、SAP IBP、PlanetTogether
